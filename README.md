# 📊 Customer Churn Prediction Dashboard

## 🚀 Project Overview

This project predicts customer churn using machine learning and provides insights through a simple dashboard.

## 🧠 Features

* Customer churn prediction using ML
* Interactive frontend UI
* Flask backend API
* Real-world telecom dataset

## ⚙️ Tech Stack

* Python (Pandas, Scikit-learn)
* Flask
* HTML, CSS, JavaScript

## ▶️ How to Run

```bash
pip install -r requirements.txt
python app.py
```

## 📌 Output

* Predicts whether a customer will churn or not
* Helps businesses take preventive actions
